I worked on a tool aimed to streamline the annotation process for chart corpora in SVG elements, enabling efficient automation of data visualization.

In this project, I wrote a Python script to convert a static webpage into an HTTP backend server that serves files, handles GET requests, and processes POST requests to save JSON data to files.

This acted as a save feature by exporting and reloading specific annotation variables to the HTTP server.
